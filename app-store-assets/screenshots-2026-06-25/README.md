# PopPath App Store Screenshots

Captured on June 25, 2026 from the local Debug simulator build.

Apple accepts 1-10 screenshots per device display set. This folder has 10 distinct screens for each required set:

- `iphone-6.9/`: iPhone 17 Pro Max simulator, `1320x2868`
- `ipad-13/`: iPad Pro 13-inch simulator, `2064x2752`

## Upload Order

Use the same order for iPhone and iPad:

1. `01-home` - main screen
2. `02-tutorial` - tutorial
3. `03-classic-game` - classic gameplay
4. `04-daily-game` - daily challenge
5. `05-paused-tools` - pause/settings tools
6. `06-result` - result screen
7. `07-records` - records and achievements
8. `08-settings` - settings
9. `09-block-guide` - block guide
10. `10-tier-ladder` - tier ladder

The first 1-3 screenshots are the most important because App Store search and preview surfaces can show only the first few screenshots.
